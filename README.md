# Chrome Extension MVP — Reuso de Conteúdo

Estrutura inicial do MVP de extensão Chrome (Manifest V3) usando React + TypeScript + Vite.

Objetivo: ajudar candidatos a reutilizar snippets e currículos durante processos seletivos.

Estrutura criada como esqueleto; funcionalidade será implementada posteriormente.

Como rodar (após instalar dependências):

```bash
npm install
npm run dev
```
